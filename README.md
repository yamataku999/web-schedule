# 📋 WEB制作 スケジュール管理システム

Webディレクター向けのスケジュール管理ツールです。
複数案件をタブで管理し、ガントチャートで進捗を可視化できます。

---

## 機能

- **個人アカウント** — 名前と4桁の暗証番号でログイン。他の人のデータは見えません
- **複数案件管理** — タブで案件を切り替え、何件でも追加可能
- **ガントチャート** — 工程ごとのスケジュールを自動で可視化
- **デッドライン管理** — 公開希望日との差分を自動計算
- **バッファ設定** — 全体工期にバッファを自動加算
- **データ保存** — ブラウザに自動保存（ページを閉じても消えない）

---

## 使い方

### 初回（アカウント作成）
1. URLにアクセス
2. 「新規アカウントを作成する」をクリック
3. 名前と4桁の暗証番号を登録

### 2回目以降
1. URLにアクセス
2. 名前と暗証番号を入力してログイン
3. 前回の案件データがそのまま表示されます

---

## ⚠️ 注意事項

- データは**各自のブラウザ**に保存されます
- ブラウザのキャッシュを消去するとデータが消えます
- 別のPCや別のブラウザからは同じデータにアクセスできません
- 暗証番号を忘れた場合はリセットできません（管理者に連絡）
